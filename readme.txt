== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/launch6-script-plugin` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Click on the Launch6 icon found in the left-hand navigation, and paste your Launch6 Code in the box provided and click Save.
